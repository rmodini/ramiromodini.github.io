WORK

Learn Japanese Kana


Project built with React Native and available now on the Play Store and soon on the Apple Store. Educational mobile application to learn to read and understand Japanese hiragana and katakana! With a very effective card swiping system, divided in two parts: first memorize and then practice (take a quiz) you are going to learn to read and understand Japanese kana perfectly in less than 2 weeks. Includes a very useful Resources section where you can find everything you need to know to start learning Japanese from scratch (tables, diacritics, phonetic marks, and more!)

BuscaMate


Project that uses the Google Maps API and PostgreSQL database. Users can look for yerba mate selling locations anywhere, anytime! Many migrants, specially from Latin America, have the problem that they cannot quickly and effectively locate a store to buy yerba mate near their new city. This app is here to solve that problem! It allows users to locate yerba mate selling points anywhere in the world, as well as the option to add a new selling location if it is not already on the map and making the database bigger and better to everyone. Available now on the Play Store and soon on the Apple Store.



View on Google Play

BandApp


Project in development built with React Native and Firebase, using Google Firestore Cloud Services to store data live. Users can create an account and login, and then post an ad looking for a band or musician. The ad will render on the main post feed for everyone to see. Users can here filter the posts by city. They can send a private message to the user that posted the ad and begin a live conversation there. All current chats will be saved on the chats screen to continue the conversation. There are lots of more features to be added! Stay tuned for the release!
